### Type Hint | Dica

*Foi implementado somente na versão 3.5 do Python. Portanto, é importante utilizar o pyenv para garantir que a versão do Python seja igual ou superior a 3.5*


Python também é uma linguagem de *tipagem forte*, o que significa que o interpretador não realiza conversões automáticas entre tipos diferentes.


 - A tipagem do Python é feita por inferência, ou seja, o interpretador do Python deduz o tipo de uma variável com base no valor atribuído a 
 ela. Diferente de linguagens como Java ou C++, onde é necessário declarar explicitamente o tipo de uma variável, no Python você simplesmente atribui um valor e o tipo é automaticamente determinado.
 
 - É uma tipagem dinâmica, o que significa que o tipo de uma variável pode mudar durante a execução do programa. Por exemplo, você pode 
 
 - inicialmente atribuir um número inteiro a uma variável e, em seguida, atribuir uma string à mesma variável sem nenhum erro.


#### *A partir deste momento do bootcamp, por padrão, para  facilitarmos a leitura do código, vamos declarar o tipo das variáveis utilizando type hints.*
